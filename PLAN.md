# 使用 uv 建立本機 PDF 合併工具

## 摘要

使用 Python 3.13、Streamlit 與 pypdf 建立本機網頁工具。使用者可上傳多個 PDF、調整順序、指定輸出檔名，並下載合併結果。文件只在記憶體中處理，不上傳外部服務、不永久保存。

## 實作內容

- 使用 uv 管理 Python、虛擬環境、正式及開發依賴，提交 `pyproject.toml`、`.python-version` 與 `uv.lock`。
- 建立獨立 PDF 合併模組，驗證檔案後以 `PdfWriter.append()` 依畫面順序合併，輸出 `BytesIO`。
- Streamlit 介面支援多檔上傳、頁數顯示、上移、下移、移除、清除、輸出命名及下載。
- 損壞、非 PDF、空白或加密檔案會中止整批合併，且不產生部分結果。
- 不寫入磁碟暫存，不提供壓縮、裁切、頁面範圍或密碼輸入。

## 測試與驗收

- 測試順序、總頁數、頁面尺寸與方向、中文及重複檔名。
- 測試少於兩份、空白、損壞、非 PDF 與加密 PDF。
- 執行 `uv run pytest`，核對代表性輸出的頁數並渲染頁面進行視覺檢查。
- 執行 Streamlit smoke test，確認應用程式可啟動。

## 限制

第一版僅供本機單一使用者。主要保留頁面視覺內容，不承諾保留書籤、附件、互動表單、數位簽章或文件層級中繼資料。
